What is PerlDB?

PerlDB is a light weight relational database engine built entirely in Perl programming language.

Primary Features

  1. Create, rename and delete table
  2. Add, rename, and delete column
  3. Insert, update, delete row
  4. Primary key, foreign key constraints
  5. Referential integrity: update restricted and delete restricted
  6. Customised table triggers
  7. Basic transaction and concurrency management
  8. Transaction log
  9. Shared lock and exclusive lock on table
  10. Relational algebras: Select, Project, Cross, Join, Redefine

Highlights

  1. Pure Perl implementation
  2. Data files are well-formatted to be human readable
  3. Thus, can easily be processed by any text processing language
  4. Tiny memory print: almost all table operations are done on disk
  5. Well documented and commented code
  6. Very compact design and thus very small in size

When do you need PerlDB?

PerlDB is very compact and easy to understand, thus it is an ideal choice when you need to customize some behaviors of a relational database engine. It offers many functions which are not seen on other similar sized DB engines, such as human-readable data file, customized table triggers and support for transaction and concurrency management.
